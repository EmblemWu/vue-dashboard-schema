#+TITLE: Schema Dashboard User Guide

* 页面说明
- /#/templates: 模板列表（可收藏、可进入详情/大屏）
- /#/templates/:schemaKey: 模板详情（查看组件清单）
- /#/favorites: 收藏管理
- /#/screen/:schemaKey: 大屏渲染页

* 两条可跑通路径
1. 模板列表 -> 模板详情 -> 打开大屏
2. 模板列表收藏 -> 收藏管理 -> 打开大屏

* 本地运行
#+begin_src bash
pnpm install
pnpm dev
#+end_src

* 质量门
#+begin_src bash
pnpm lint
pnpm typecheck
pnpm test
pnpm build
pnpm test:e2e
#+end_src

* 新增模板
1. 在 =schemas/= 添加 JSON 文件
2. 按 =docs/SCHEMA.md= 定义字段
3. 在模板列表查看是否加载成功

* 异常与兜底
- 页面级：加载态/空态/错误态/重试
- 全局：ErrorBoundary 捕获运行时异常
- 图表资源失败时降级显示提示文案
